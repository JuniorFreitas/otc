Como Calcular

1. Métricas Básicas

Cobertura = Paradas ÷ Área
Capacidade = (Lugares × Viagens × Km) ÷ População
Frequência = Viagens

2. Normalização

UX = (Valor - Média) ÷ Desvio Padrão

3. Índice OTC

OTC = UX Cobertura + UX Capacidade + UX Frequência

4. Níveis

A Excelente (≥ 2.0)
B Bom (1.0 - 2.0)
C Regular (0.0 - 1.0)
D Ruim (-1.0 - 0.0)
E Muito Ruim (-2.0 - -1.0)
F Péssimo (< -2.0)
